# 🚚 Supply Chain and Logistics Analytics Using SQL

## 📖 Project Overview

This project demonstrates how SQL can be used to solve real-world supply chain and logistics business problems. Using a relational database built in MySQL, the analysis explores transportation performance, warehouse utilization, freight costs, customer demand, route optimization, and operational efficiency through business-driven SQL queries.

The objective is to transform raw operational data into actionable business insights that support decision-making across logistics, warehouse management, procurement, and transportation planning.

---

## 🎯 Business Objectives

This project aims to answer key operational questions such as:

- Which carriers have the highest late shipment rates?
- Which shipping routes are the most expensive?
- Which warehouses exceed their daily capacity?
- Which customers generate the highest logistics demand?
- Which transportation modes provide the best cost-to-service performance?
- Which products create the highest logistics burden?
- Which regions rely heavily on a single carrier?
- What factors contribute most to shipment delays?
